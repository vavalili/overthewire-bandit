- `nmap` Network exploration and security scanning tool
- `-A` enables OS detection, version detection, script scanning, and traceroute
- `-p 31000-32000` Scans only ports 31000 through 32000
- `localhost` Scans the local machine (127.0.0.1)
- `echo "password"` Sends the current level's password as input
- `|` (pipe)	Connects the output of echo to the input of openssl
- `openssl s_client`	Opens an SSL/TLS connection to a server
- `-connect localhost:31790`	Connects to the SSL service on port 31790
- `-quiet`	Suppresses extra openssl output, showing only the response
- `nano /tmp/key17` opened a text editor to paste the RSA private key into a file called key17
- `chmod 600 /tmp/key17` This sets the file permissions so only you can read/write it this is requires from SSH for security
`ssh -i /tmp/key17 -p 2220 bandit17@bandit.labs.overthewire.org`  uses the private key to log in to Bandit Level 17 over SSH on port 2220.
